Place official logos in this folder to render in the project README.

Expected filenames (used by README.md):
- autodesk-revit.png        (Autodesk Revit logo)
- microsoft-copilot.png    (Microsoft Copilot / Copilot Studio logo)
- model-context-protocol.svg (Model Context Protocol logo)
- python.svg               (Python logo)
- dotnet.svg               (.NET logo)

Notes:
- Download logos from the official brand resources pages and respect each vendor's brand guidelines.
- Alternatively, you can hotlink logos in `README.md` by replacing the `src` attribute with an external URL.
- To add a logo from your local machine (PowerShell):

  Copy-Item -Path "C:\path\to\logo.png" -Destination "assets/logos/autodesk-revit.png"

- Commit the files to make them visible on GitHub.
